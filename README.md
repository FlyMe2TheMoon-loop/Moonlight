# 🌙 Moonlight Projects

Este es mi primer repositorio de Python corriendo en **Zorin OS Lite**. Aquí guardo herramientas útiles para automatizar mi día a día.

## 🛠 Proyectos Incluidos

### 1. File Organizer (`organizer.py`)
Un script inteligente que organiza mi carpeta de descargas automáticamente.
* **Mueve:** Imágenes, Documentos, Instaladores y Archivos comprimidos a sus carpetas correspondientes.
* **Beneficio:** Mantiene mi disco de 1TB limpio y ordenado.

### 2. Network Scanner (`network_scanner.py`)
Un escáner de red basado en el protocolo ARP.
* **Función:** Detecta qué dispositivos (celulares, laptops, TVs) están conectados a mi WiFi.
* **Requisito:** Necesita ejecutarse con `sudo`.

Para ejecutar cualquiera de estos scripts, abre tu terminal y escribe:
